# 伯洋體產生器 🎤

> 一鍵生成讓人「有聽跟沒聽一樣」的伯洋體語句，並直接輸出成 Threads 串文格式。

---

## 🗣 什麼是伯洋體？

**伯洋體**，源自 2026 台北市長選舉期間，外界對候選人發言風格的一種調侃標籤。  
其核心特徵為：**只講美好目標，不講具體做法**，常搭配抽象術語與因果串聯，讓聽眾產生「聽起來很有道理，但完全不知道在講什麼」的奇妙體驗。

### 經典公式

```
讓【對象】找回他失去的【A】
→ 有了 A，自然就會有【B】
→ 而 B 帶來的，正是【C】的根本
→ 就是那麼簡單的道理而已。
```

---

## ✨ 功能介紹

本工具提供四種公式風格，可自由切換：

| 風格 | 說明 |
|------|------|
| 🔹 **基礎版** | 標準三段式伯洋體語句 |
| 🔹 **學術煙霧彈版** | 加入「第三空間感知韌性對齊」等術語，格局瞬間提升 |
| 🔹 **反串崇拜版** | 全程以崇拜口吻反諷，殺傷力最強 |
| 🔹 **Threads 串文版** | 直接輸出 5 則串文 + hashtag，可直接貼上發佈 |

### 其他功能
- 🎲 **隨機填入**：一鍵隨機產生詞組，靈感不斷
- 📋 **一鍵複製**：輸出結果一鍵複製到剪貼簿
- 🌙 **深色 / 淺色主題**：支援手動切換

---

## 🚀 使用方式

1. 開啟 `index.html`（或透過 GitHub Pages）
2. 選擇公式風格（基礎版 / 學術版 / 反串版 / Threads 版）
3. 填入「對象、名詞 A / B / C、學術關鍵詞、政策主題」
4. 點擊「🎤 產生伯洋體」
5. 複製輸出，貼上 Threads 即可發佈
